Here is the list of features and their arguments:

**Extract Pages**
- `file`: The path to the PDF file you want to extract pages from.
- `start`: The first page number of the range to extract.
- `end`: The last page number of the range to extract.

**Insert Pages**
- `source_file`: The path to the main PDF file you are adding pages to.
- `inserted_file`: The path to the PDF file containing the pages you want to insert.
- `insert_pos`: The page number in the main PDF that will be the reference point for the insertion.
- `relative_pos`: Determines where to place the new pages. Use 'before' or 'after' the `insert_pos`.
- `start_insertion` (optional): The first page of the range to copy from the `inserted_file`. Defaults to the first page.
- `end_insertion` (optional): The last page of the range to copy from the `inserted_file`. Defaults to the last page.

**Merge Files**
- `files`: A list of paths to the PDF files to merge.

**Rearrange Pages**
- `file`: The path to the PDF file you want to reorganize.
- `start`: The first page number of the block you want to move.
- `end`: The last page number of the block you want to move. (Use the same number as `start` to move a single page).
- `relative_pos`: Determines where to place the block. Use 'before' or 'after' the `new_pos`.
- `new_pos`: The page number that will be the reference point for the move.

**Remove Pages**
- `file`: The path to the PDF file you want to remove pages from.
- `start`: The first page number of the range to delete.
- `end`: The last page number of the range to delete.
